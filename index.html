Code example

Our mission: write a function nouveau (that's French for "new") which takes one function parameter (the constructor), plus an unknown number of additional parameters of any type (arguments for the constructor). When invoked, nouveau should do everything new does and return the same object new would evaluate to, as specified above.

function nouveau(Constructor, ...args) {
  let obj = Object.create(Constructor.prototype)
  let ret = Constructor.apply(obj, args)
  return ret !== null && (typeof ret === 'object' || typeof ret === 'function') ? ret : obj
}
